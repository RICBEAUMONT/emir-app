# Fonts for Email Signature Guide

Place your web font files here so the guide can use them:

- **Miller Banner** (titles/headings): `MillerBanner.woff2` and/or `MillerBanner.woff`
- **Akkurat Regular Pro** (body text): `AkkuratRegularPro.woff2` and/or `AkkuratRegularPro.woff`

If your files have different names, update the `@font-face` `src` URLs in `index.html` to match.

If you use **Adobe Fonts** (Typekit) instead, add the embed code in the `<head>` of `index.html` and remove or adjust the `@font-face` blocks so the font-family names match your kit.
